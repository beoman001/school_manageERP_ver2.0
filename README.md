Welcome to the **Ultimate School ERP System**—a modern, responsive, and feature-rich digital campus dashboard. This application is built with a highly flexible design engine featuring three distinct, selectable user experiences: **Friendly EdTech (Soft UI)**, **Deep Space Dark**, and an electric **Neon Green Hacker Mode**.

## 🎨 Design System & Core Themes

This application moves away from boring, rigid spreadsheets and utilizes a physics-based animation system alongside a 3-way fluid theme controller:

* **Friendly EdTech (Default):** Vibrant indigo accents, generous rounded borders, and bouncy spring physics inspired by modern learning apps.
* **Deep Space Dark:** High-contrast slate and deep purple panels engineered for comfortable night-time administration.
* **Neon Green:** An electric, ultra-modern layout with glowing borders and sharp cyber aesthetics.

---

## ⚡ Core Features

### 👤 Role-Based Portals
* **System Admin:** Command center featuring live campus statistics, recent system logs, and quick data directives.
* **Faculty/Teacher:** Bulk attendance logging, exam marks entry matrix, and study material distribution pipelines.
* **Student:** Interactive visual attendance calendars, direct assignment download desks, and instant grade monitoring.
* **Parent:** Secure synchronization with child credentials to view real-time performance tracking and fee statuses.

### 🛠️ Advanced Modules
* **Auto-Generated Report Cards:** Custom `@media print` layout engine that formats student grades into a printable, professional A4 PDF.
* **Secure Chat Engine:** WhatsApp/iMessage-style layout allowing real-time chat between students, parents, and teachers with one-click conversation wiping.
* **Smart Transport Calculator:** Zone-based transport distance fee structure validator.

---

## 🛠️ Tech Stack

* **Backend:** Python 3.10+, Flask
* **Database Object Mapper:** Flask-SQLAlchemy (SQLite for development)
* **Security:** Werkzeug Password Hashing (`scrypt`)
* **Frontend UI:** Vanilla HTML5, Custom CSS3 Variables, Native JavaScript Theme Engine
